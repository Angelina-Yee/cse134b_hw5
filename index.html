Projects

Triton Circles

  • A class group project, consisting of five members, with the main objective of developing a web application to encourage UCSD students’ involvement in student organizations and clubs at UCSD.
  • The web application is developed mainly using React and SQlite.
  • Students and users are able to create a profile by signing in (and logging in) and answering a questionnaire that personalizes their profile.
  • The main features for the web application addresses several user stories about finding new clubs, applying for board positions, receiving calendar notifications for club events, verifying attendance for clubs, viewing friends’ possible attendance for events, and the ability to hide activity (attendance) from friends.

Nom-Pal

  • A personal project, with a partner, mainly using HTML and CSS for front-end practices.
  • A front-end project for a future web application that allows users to log items in their pantry or fridge and search for recipes that they have added according to the items and the amount they have.
  • Features available currently includes: profile image change choices, sign up and log in. Pages available currently includes: home page, profile page, contact us page, and about page.
  • Application will have more features added as time progresses and we utilize JavaScript, and other back-end properties.
